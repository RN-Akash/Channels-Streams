# Privacy Policy


## 1. Introduction
This project (Channels Streams) is an open-source application that provides access to media streams using third-party URLs.

## 2. No Data Collection
We do not collect, store, or share any personal information from users.

This includes:
- No user accounts
- No login system
- No personal data storage

# 3. Third-Party URLs
This application uses third-party streaming URLs to display content.

Important points:
- We do not own or host any of these streams
- All content is provided by external sources
- These third-party services may collect data independently
- We are not responsible for how third-party URLs handle user data.

## 4. No Control Over External Content
Since all streams come from external providers:

- We do not control availability, quality, or legality
- Content may change or be removed at any time
- Users are responsible for how they access and use the content.

## 5. Security
We do not store user data, so:

- There is no internal risk of personal data leaks
- However, third-party links may have their own risks

## 6. Children’s Privacy
This project does not knowingly collect any data from children.

## 7. Changes to This Policy
This Privacy Policy may be updated at any time. Changes will be reflected in this document.
